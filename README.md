# Kuznecov.io

Моё резюме на Vue (Composition API), Apollo (GraphQL), Scss, Nuxt-ts, Express — [kuznecov.io](https://kuznecov.io/)


## Build Setup 

``` bash
# install dependencies
npm install # or yarn

# serve in dev mode
yarn dev

# build for production
yarn build

# serve in production mode
yarn start
```
